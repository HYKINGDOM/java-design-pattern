# super-portal

#### Description
基于Spring Cloud微服务架构，支持配置中心、注册中心、网关，通过OAuth 2.0协议进行授权，并利用JWT实现用户信息的传递。
网关采用Spring Cloud Gateway为基础，动态根据注册中心扫描实现路由注册。配置中心使用的是本地文件方式进行共享。
主要模块包括：注册中心、配置中心、网关、用户中心、鉴权中心、文件存储服务、微信接入和公众号管理、内容管理等。
